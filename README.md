# 🛍️ VapeShop Bot v2

Telegram бот с Mini App для продажи жидкостей, подов и расходников.

## ✨ Возможности v2

### Для пользователей:
- 🛒 **Каталог товаров** с категориями
- 📱 **Mini App** в стиле Apple/Instagram
- 🛍️ **Корзина** с управлением количеством
- 📝 **Оформление заказа** с доставкой
- 👤 **Профиль** с историей заказов

### Для администраторов:
- 📊 **Статистика** магазина
- 📦 **Управление заказами** (статусы)
- 🛍️ **Добавление товаров** с фото через Telegram
- 👥 **Список пользователей**

## 🚀 Быстрый старт

### 1. Настройка .env
```
BOT_TOKEN=ваш_токен
ADMIN_IDS=ваш_id
WEB_APP_URL=https://your-app.railway.app
PORT=8080
```

### 2. Деплой на Railway
1. GitHub репозиторий подключен
2. Railway → New Project → Deploy from GitHub
3. Добавить переменные окружения из .env
4. Деплой автоматический

### 3. Настройка BotFather
- Bot Settings → Menu Button → Configure Menu Button
- URL: `https://your-app.railway.app`
- Название: `🛒 Открыть магазин`

## 🛍️ Добавление товара (админ)

1. В боте: **🛍️ Товары** → **➕ Добавить товар**
2. Отправьте по шагам:
   - Название
   - Описание
   - Цена (число)
   - ID категории (1-4)
   - Фото (или "пропустить")

## 🎨 Mini App

Дизайн в стиле официального магазина:
- Чистый минималистичный интерфейс
- Сетка товаров 2 колонки
- Плавные анимации
- Корзина с модальным окном
- Оформление в 1 клик
